🛍️ Customer Shopping Behavior Analysis
📌 Overview

This project analyzes customer shopping behavior to understand purchasing patterns, customer preferences, and sales trends.

The objective is to transform raw retail data into actionable insights that help businesses improve customer targeting, increase sales, and optimize decision-making.

This is a complete end-to-end data analytics project covering Python, SQL, and Power BI.

📂 Dataset
File Name: customer_shopping_behavior.csv
Contains transactional and customer-related data
🔑 Key Columns:
Customer ID – Unique customer identifier
Age, Gender – Customer demographics
Item Purchased – Product bought
Category – Product category
Purchase Amount – Transaction value
Location – Customer location
Payment Method – Mode of payment
Frequency of Purchase – Buying behavior
Review Rating – Customer satisfaction
🛠️ Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib, Seaborn) → Data Cleaning & EDA
SQL (MySQL Workbench) → Querying & Business Analysis
Excel → Initial Data Inspection
Power BI → Dashboard & Visualization
Gamma → Presentation (PPT)
🔄 Project Workflow
1. Data Loading (Python)
Loaded dataset using Pandas
Inspected structure, null values, and data types
2. Data Cleaning
Handled missing values
Removed duplicates
Standardized categorical values
Converted data into proper formats
3. Exploratory Data Analysis (EDA)
Analyzed customer demographics (age, gender)
Identified top-selling products and categories
Studied purchase behavior trends
Evaluated customer ratings and satisfaction
4. SQL Analysis (MySQL Workbench)
Performed queries to extract insights:
Total revenue by category
Average purchase amount by gender
Customer segmentation
Most frequent buyers
Used GROUP BY, JOIN, AVG(), SUM(), COUNT()
5. Power BI Dashboard
Built interactive dashboard including:
📊 Sales Overview KPIs
🛒 Category-wise Sales
👥 Customer Demographics
📍 Location-based Analysis
⭐ Customer Ratings Insights
6. Reporting & Presentation
Created business insights report
Designed presentation using Gamma
Highlighted trends and recommendations
